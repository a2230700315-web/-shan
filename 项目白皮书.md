## 一、项目概述

### 1.1 项目定位
闪吊平台是一个连接无人机吊装服务需求方和服务提供方的撮合平台，采用**纯前端演示模式**，所有数据存储在浏览器 LocalStorage 中，暂无后端服务。

### 1.2 技术栈现状
| 层次 | 需求规划 | 当前实现 |
|------|----------|----------|
| 前端框架 | Vue 3 + Element Plus | ✅ React 18 + TypeScript + TailwindCSS |
| 小程序端 | 微信小程序原生 | ❌ 未实现 |
| 后端框架 | Spring Boot | ❌ 未实现 |
| 数据库 | MySQL | ❌ 使用 LocalStorage |
| 缓存层 | Redis | ❌ 未实现 |
| 文件存储 | MinIO | ❌ 前端模拟 |
| 实时通信 | WebSocket/SSE | ❌ 未实现 |
| 消息推送 | 微信订阅消息 + 短信 | ⚙️ 接口预留 |

## 七、技术债务

1. **数据持久化**：当前使用 LocalStorage，数据量有限，无法多设备同步
2. **文件上传**：前端模拟，无真实存储
3. **定时任务**：超时处理逻辑已实现，但无后台调度
4. **安全认证**：无真实登录验证，无HTTPS
5. **测试覆盖**：仅有少量单元测试，缺乏E2E测试
